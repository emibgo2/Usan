이용 안내

USAN에 대한 사용 방법입니다.

대여 방법

  1. 보관함에 있는 QR코드를 인식하세요.
  2. 사이트에 접속하여 로그인 후 대여하기 버튼을 눌러주세요.
  3. 우산을 인식존에 인식해주세요.
  4. 이후 안내에 따라 대여를 진행해주세요.
  5. 결제 후 레일을 따라 우산을 꺼내주세요.

반납 방법

  1. 보관함에 우산을 넣어주세요.
  2. 우산을 인식존에 인식해주세요.
  3. "띠리링" 소리가 난다면 그대로 넣어주세요.
  4. 만약 추가요금이 있다면 결제가 잘 됐는지 확인해주세요.

* 만약 "띡" 소리가 난다면 당황하지 마시고 천천히 다시 시도해주세요.

요금 안내

USAN 사용 요금 안내입니다.

  1. 대여한 날짜까지의 기본 우산 사용 요금은 700원 입니다.
  2. 대여시간으로부터 24시간 경과 후 일당 400원 추가요금 발생

ex) 3일 대여 후 2일 연체시
(기본 우산 사용 요금 : 700원, 연체 하루당 금액 : 400원)
2일 연체시 금액 : 1,500원
