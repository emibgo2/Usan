이용 안내

USAN에 대한 사용 방법입니다.

대여 방법

  1. 보관함에 있는 QR코드를 인식하세요.
  2. 사이트에 접속하여 로그인 후 대여하기 버튼을 눌러주세요.
  3. 이후 안내에 따라 대여를 진행해주세요.
  4. 대여완료 후 우산을 레일 방향에 따라 꺼내주세요.

반납 방법

  1. 보관함에 반납할 우산을 넣어주세요.
  2. 삑소리가 난다면 레일 방향을 따라 넣어주세요.
  3. 우산 반납에 성공했다면 결제가 잘 됐는지 확인해주세요.

* 만약 "인식을 다시 해주세요" 라는 음성 안내가 나온다면 당황하지 마시고 천천히 다시 시도해주세요.

요금 안내

USAN 사용 요금 안내입니다.

  1. 대여한 날짜까지의 기본 우산 사용 요금은 0000원 입니다.
  2. 정상 반납시 기본 환급 금액은 0000원 입니다.
  3. 연채시 하루당 기본 환급 금액의 000원 만큼 차감합니다.
  4. 연채 후 반납 완료시 돌려받는 환급 금액이 줄어듭니다.

ex) 3일 대여 후 2일 연채시
(기본 우산 사용 요금 : 2,000원, 연채 하루당 금액 : 500원)
2일 연채 후 환급 금액 : 2,000원 - (500원 X 2일) = 1,000원
